About Me

I've been working in industry as a Junior Web Developer for a few years now, and before that I was studying Computer Science at the University of Brighton where I graduated in 2020! Object oriented programming, relational database structures, and project management are some of the key areas I was educated in.

Before my studies, I spent a lot of time creating static pages with HTML and CSS, and In hindsight, this is where my web developer journey really started as I began to develop a passion for UI/UX design.

As for the future, what excites me most about web development is having the ability to bring innovative concepts to life that can create a positive user experience, and as a web developer still learning their craft, this is ultimately how I see myself utilising my skills.
